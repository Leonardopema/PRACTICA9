Importancia del Bootstrap

Bootstrap es un conjunto de herramientas de código abierto muy popular entre los expertos en desarrollo web, ya que está ideado para un desarrollo responsive. Es decir,gracias a Bootstrap, los desarrolladores web pueden crear páginas web visibles en diferentes formatos y pantallas. El objetivo principal de Bootstrap es crear sitios web responsivos y orientados a los dispositivos móviles. El framework garantiza que todos los elementos de la interfaz de un sitio web funcionen de forma óptima en todos los tamaños de pantalla.
